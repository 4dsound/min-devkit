# convolve~

Perform convolution on a list.

Convolution is...

## Messages

* `List` : Input to the convolution function.

## Attributes

* `kernel` : The convolution kernel.  
             The kernel is the operand of the convolution function.
             
## Output

A list of the same dimensions as the input list.

## See Also

* `buffir~` : Perform convolution on an audio signal using a buffer~ as the kernel.
* `jit.convolve` : Convolve two matrices.
